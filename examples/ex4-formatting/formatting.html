Formatting Elements

HTML5 provide different formatting tags which, by default, have some formatting function, but that are rather associated to the semantic meaning of the text in the page.

Strong and Bold

The strong tag is used to mark the importance of something.

It also possible to use the b tag, which is meant just to make some text bold, without any semantic reference.

Example

If you see a car with license plate GE 007 please contact the police

Emphasize and Italic

The em tag is used to emphasize a text

It is also possible to use the i tag, which is meant just to make some text italic, without any semantic reference.

Example

Bonjour is the sexiest greeting ever existed!

Highlight

The mark tag is used to highlight a part of text.

Subscript and Superscript

It is possible to add sub- and super-script to text with the sub and sup tags.

Example

With sub- and super-script we can easily create fractions 1/2.

Insertion and Delete

The ins tag allows to define text that has been inserted.

On the other hand, the del tag defines text that has been (or should be) deleted.

Example

The next STIC I course is next Monday Friday!
