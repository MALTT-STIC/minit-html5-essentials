Text content

The organization of textual content is a useful way to facilitate reading. It can make the page more easy to scan for important information, or provide a structure that facilitate the relation between contents.

Headings

Headings have multiple levels of importance.

h1: Top level heading

h2: Second level heading

h3: Third level heading

h4: Forth level heading

h5: Fifth level heading (rarely used)
h6: Sixth level heading (rarely used)

Paragraphs

This is a paragraph of text, which is intended to contain one or more text sentences, usually describing a concept. The paragraph tag is widely used.

This is a second paragraph, which is also intended to contain some text.

Unordered Lists

Unordered lists use the

    tag to start the list, and the
  • tag for each element of the list. Here is an example.

    A good HTML page comprises at least:

    • A meaningful title
    • Links that work!
    • Text carefully organized in paragraphs

    Ordered lists

    Ordered lists use the

      tag to start the list, and the
    1. tag for each element of the list. Here is an example.

      The 5 biggest countries in the world are (source Wikipedia):

      1. Russia
      2. Canada
      3. China
      4. USA
      5. Brazil
