Different kinds of links

All links are created with the tag "a" which supports many attributes. The two most important attributes are:

  • href which determines where the link will point Text of the link
  • target which determines where the linked page will be shown (by default in the same tab/windows) Text of the link opening in a new tab/windows (which is better to avoid)

There are basically three kinds of links, the type depending on the value of the href attribute:

  1. Absolute links
  2. Relative links
  3. Bookmark (or anchor) links

Absolute links

Absolute links takes the whole URL of a page as value of the href:

  • http://tecfa.unige.ch/ --> TECFA
  • https://developer.mozilla.org/en-US/docs/Learn/HTML/Introduction_to_HTML/Creating_hyperlinks --> HTML links on MDN

Relative links

Relative links refer to pages that are part of the same domain and thus do not require the whole URL of the page, but just the path relative to the current position:

  • If the page is in the same directory, the filename is enough: links-second-page.html --> Link to a second page in same directory
  • If the page is in a subfolder, the name of the folder and the filename are required: subfolder/links-third-page.html --> Link to a third page in subfolder
  • If the page is in a parent or sibling directory, we can use the ../ notation which stands for "one level up": ../ex1-typography/typography.html --> Link to the typography example

Warning: the wrong path of links is one of the most common mistakes for beginners!

Bookmark links

Bookmark links are internal to the same page and are useful when the page has a lot of content. In order to use bookmark links, you must provide the target element with an id="unique_id" attribute and set the href of the link to href="#unique_id":
