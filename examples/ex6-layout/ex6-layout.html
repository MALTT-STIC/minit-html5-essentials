Layout

The header should contain information such as the logo/title of the site/page

Navigation

The nav should contain the main menu of the site.

  • Home
  • About
  • Projects
  • Contacts

Aside

The aside should contain contextual information to the main content.

Main

The main element should contain the content of the page, that is the content that is specific to this page. This is where you may use elements such as article and section.

This is a blog post.

The article can be divided into sections.

Introduction

Text for the introduction

Development

Text for the section.

Conclusion

Text for the conclusion.

Footer

The footer should contain general information, such as the © of the page, the author, the date, etc.
