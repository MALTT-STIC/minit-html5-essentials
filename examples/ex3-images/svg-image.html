SVG

Scalable Vector Graphics (SVG) differs from the images presented in the other file since they are vectors, that is, they are computed by mathematic formulas. As a consequence, they maintain the same shape and quality regardless of the size and the resolution of the screen.

Comparison SVG vs. PNG

Here you can see the difference. The original .png image is 50x50 pixels. Notice how the quality deteriorates when scaled up.

On the contrary, SVG scales without any loss of quality

How to include SVG

There are 5 different ways to include SVG files in an HTML5 page :

  1. Inline, with the svg tag (see source of the previous example)
  2. Using the img tag as for images, giving a src attributes that points to the svg external file.
  3. Using the embed tag
  4. Using the object tag
  5. Using an iframe

In this page we will show just the img example. For more information, refer to the EduTechWiki page on static SVG.
