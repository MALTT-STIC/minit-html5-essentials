Tables

Tables have been used in the past in order to create page layouts. This practice is now highly discouraged, and tables should be used only to organize data.

Tables have multiple options and can be customized in many ways. The basic elements are:

  • The <table> tag wraps up all the content
  • The <tr> tag stands for a row of the table
  • The <td> tag creates a normal cell of a column into a row
  • The <th> tag creazes an heading cell of a column into a row
  • The <thead> and <tbody> tags separate the headings and the content of the table
